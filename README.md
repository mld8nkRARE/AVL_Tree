# AVL_Tree
Учебный проект: реализация AVL-дерева на C# и сравнение его производительности с `SortedDictionary<TKey, TValue>`.
## Структура проекта
| Проект | Описание |
|---|---|
| **AVL_TreeLib** | Структура данных `AVL_Tree<TKey, TValue>` |
| **ExperimentsConsoleApp** | Бенчмарк |
| **AVL_TreeTest** | Юнит-тесты для класса `AVL_Tree<TKey, TValue>` |
## AVL-дерево
Самобалансирующееся дерево поиска.
- Вставка — `O(log n)`
- Удаление — `O(log n)`
- Поиск — `O(log n)`
## Бенчмарк
Генерирует дерево из 100 000 случайных чисел, удаляет 1/4 чисел, выполняет поиск всех чисел в дереве.
| Структура данных | Время |
|---|---|
| AVL Tree | 209 ms |
| SortedDictionary (.NET) | 182 ms |
